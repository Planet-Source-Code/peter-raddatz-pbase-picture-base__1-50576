Welcome to PBase

What is PBase?

PBase is a simple program that will display thumbnails of all *.bmp, *.jpg and *.gif    files in any directory or folder, a larger view of any of these thumbs and, if present, a description of the displayed image.

 
How does it work?

At the start of the program you are presented with the 'Open File' dialog box. Use your mouse to navigate to the desired drive and directory. If there are any images in that folder you will see their filenames displayed in the right hand panel. If these are the images that you wish to display press <Ok>, otherwise repeat this process until the desired images are located.

PBase, by default, will display the image that is in the upper left-hand corner of the screen enlarged and will load the corresponding description underneath that picture. If you wish to enlarge that picture once again simply click on it and its size will increase to the maximum size that can fit on the screen while still maintaining the proper aspect ratio. Click on the large image again and it will shrink to its former size.

Switching pictures.

Simply click on any of the displayed thumbnails to enlarge that image and to load its description (if available).
Adding / Editing an image's description.

Adding or editing the description for the currently displayed picture simply click on the <Edit Description> menu item. If there is no description available you may start typing to add one, or, if one is already present you can edit it to suit your needs. All, normal, editing features are available.

Leaving the edit box, by clicking on another thumbnail or <Next Thumbs>/<Prev. Thumbs> will, automatically, save the displayed text.
Displaying the previous/ next 10 thumbs.

If the <Next Thumbs> & <Prev. Thumbs> buttons near the bottom of the screen are active, you may click on them to display the next (up to) 10 images or to redisplay the previous 10.
Opening a different folder.

Click on the <File> menu item and select <Open>. The 'Open File' dialog box will reappear to offer you to look for images in other directories. Proceed as you did as before, clicking <Ok> when you are satisfied with your choice or <Cancel> if you changed your mind.
Quitting PBase.

Click on the <File> menu item and choose <Exit>.
Technical stuff.

This program was written to look good on a 800*600 screen, if you want to display it on any other size screen you need to amend the code to take care of that.
